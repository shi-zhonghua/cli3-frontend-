##git提交流程
```
git status -s 查看当前仓库状态
1.代码格式化:npm run precommit  代码格式化
2.git add .   添加仓库   . 提交全部的修改
3.git commit -m ""   提交本地仓库
4.git pull -r    获取仓库最新代码
5.git push origin master   提交本地代码到git仓库

```
